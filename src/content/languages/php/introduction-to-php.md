## What is PHP?

**PHP** (Hypertext Preprocessor) is a widely-used, open-source server-side scripting language. It was specifically designed for web development but is also used as a general-purpose language. Today, it powers over **75% of the internet**, including platforms like **WordPress**, **Wikipedia**, and **Facebook**.

```php
<?php
  echo "Hello, Nocturnal Codex!";
?>
```

## Why Learn PHP?

1.  **Web Supremacy**: It is the backbone of the CMS world (WordPress, Drupal, Joomla).
2.  **Modern Frameworks**: **Laravel** and **Symfony** have transformed PHP into a modern, professional ecosystem.
3.  **Fast & Efficient**: PHP 8.x is significantly faster than older versions and handles massive traffic with ease.
4.  **Massive Market**: A huge percentage of freelance and enterprise web work still revolves around PHP.

## How it Works

Wait! PHP is a **Server-Side** language. This means the code runs on the web server (like Apache or Nginx), generates HTML, and sends that HTML to your browser. The user never sees your actual PHP code.

## The Evolution to PHP 8

For years, PHP was criticized for being messy. However, with the release of **PHP 7** and now **PHP 8**, the language has gained:
-   **Strong Typing**: Catch errors early.
-   **JIT Compilation**: Blazing fast performance.
-   **Attributes & Named Arguments**: Modern, clean syntax.

> [!TIP]
> Always use the latest version of PHP (8.2+). Modern PHP is a joy to work with and bears little resemblance to the "spaghetti code" of the early 2000s.
