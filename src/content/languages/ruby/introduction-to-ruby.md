## What is Ruby?

**Ruby** is a dynamic, open-source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write. It was created in Japan in the mid-1990s by **Yukihiro "Matz" Matsumoto**.

```ruby
puts "Hello, Nocturnal Codex!"
```

## The Philosophy of Ruby

Wait! Ruby's design is guided by a unique principle: **"Developer Happiness."** Matz designed Ruby to be "human-oriented" rather than "machine-oriented." He wanted a language that developers would *love* to use every day.

## Why Choose Ruby?

1.  **Elegant Syntax**: Ruby looks almost like English. There are no semicolons and very few curly braces.
2.  **Productivity**: You can build complex features in Ruby with very few lines of code.
3.  **The Rails Factor**: **Ruby on Rails** is the world-famous web framework that powers sites like **GitHub**, **Shopify**, and **Airbnb**.
4.  **Community**: One of the most friendly and helpful communities in the programming world.

## "Everything is an Object"

In Ruby, literally **everything** is an object. Even a simple number or a boolean value has methods you can call.

```ruby
-5.abs # Returns 5
"hello".capitalize # Returns "Hello"
3.times { puts "Hi" } # Prints "Hi" three times
```

> [!TIP]
> Ruby is the perfect language for beginners who want to see results fast, but it is deep enough to power some of the largest companies in the world.
