## What are Meta Tags?

**Meta Tags** are snippets of code that tell search engines and browsers important information about your web page. They live inside the **`<head>`** of your HTML document and are not visible to users.

```html
<head>
  <meta name="description" content="A comprehensive programming roadmap for the modern developer.">
  <meta name="keywords" content="coding, roadmap, java, python, web development">
</head>
```

## The Title Tag

Wait! The **`<title>`** is the single most important SEO element on your page. It shows up in the browser tab and as the main link in Google search results.

## Social Media Optimization (Open Graph)

Wait! Have you ever noticed that when you share a link on Discord, Slack, or Twitter, it shows a beautiful image and a summary? This is done using **Open Graph** meta tags.

```html
<meta property="og:title" content="Nocturnal Codex">
<meta property="og:description" content="Become a Master Developer.">
<meta property="og:image" content="https://example.com/banner.jpg">
```

## Charset and Viewport

-   **`charset="UTF-8"`**: Tells the browser to support all international characters and emojis.
-   **`name="viewport"`**: Makes your site look good on mobile devices (Mandatory!).
