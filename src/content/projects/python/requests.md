### Why contribute to Requests?
Requests is the most downloaded Python package. It's famous for its "for Humans" API design philosophy. It's a great project to study if you want to learn how to design intuitive APIs.

### Contribution Tips
- **Bug Fixes**: Look for small, reproducible bugs.
- **Typo Hunting**: Documentation typos are a safe first entry point.
- **Compatibility**: Help with testing against different Python versions.
