### Why contribute to Scikit-learn?
If you are interested in Machine Learning, Scikit-learn is the gold standard library. Contributing here will help you understand the mathematical foundations of ML algorithms and high-performance Python programming.

### Contribution Tips
- **Focus on Examples**: Improving the examples in the documentation is highly valued.
- **Style**: They follow strict coding standards. Use `lint` tools provided in the repo.
- **Math**: Don't be afraid of the math, but start with structural or documentation issues first.
