### Why contribute to Discourse?
Discourse is one of the largest and most well-maintained open-source Rails applications. It's a great place to see how a "real" application is built at scale, including its use of PostgreSQL, Redis, and a custom plugin system.

### Contribution Tips
- **Plugins**: Writing a small plugin for a new feature is a great way to learn the codebase.
- **Frontend**: Discourse uses Ember.js on the frontend. If you know JS, you can contribute to the UI.
- **Localization**: Help translate the platform into more languages.
- **API**: Improving the REST API is a common task.
