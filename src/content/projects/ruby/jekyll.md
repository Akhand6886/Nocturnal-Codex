### Why contribute to Jekyll?
Jekyll is simple, focused, and used by GitHub Pages. It's an excellent project for beginners to learn how to manipulate the file system, process Markdown, and use the Liquid templating engine.

### Contribution Tips
- **Plugins**: Jekyll has a simple plugin system. Adding a new "Tag" or "Filter" is very straightforward.
- **Themes**: Help improve the default themes or add support for modern CSS features.
- **Bug Fixes**: There are often small bugs related to edge cases in Markdown parsing.
