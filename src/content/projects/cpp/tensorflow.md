### Why contribute to TensorFlow?
TensorFlow's core is written in highly optimized C++. Contributing here is for those who want to build the future of AI. You'll learn about graph compilation, hardware kernels (CUDA), and large-scale distributed systems.

### Contribution Tips
- **Kernels**: Look for issues involving individual operation implementations.
- **Error Messages**: Improving the clarity of core C++ error messages.
- **Community Modules**: Help with the C++ side of SIG (Special Interest Group) projects.

### Why contribute to nlohmann/json?
This is a header-only C++ library that is famous for its "first-class citizen" JSON API. It's an excellent project to study modern C++ patterns (C++11/14/17/20) in a focused, high-quality codebase.

### Contribution Tips
- **Fuzzing**: Helping to fix issues found by the fuzzer.
- **Documentation**: Improving the documentation examples.
- **Build Integration**: Help with package manager integrations (Vcpkg, Conan).
