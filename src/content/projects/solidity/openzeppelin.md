## Why Contribute?
OpenZeppelin is the standard library of the blockchain world. Almost every serious smart contract uses it. Contributing here helps improve the security of the entire ecosystem.

## Contribution Tips
- Security is the #1 priority. Expect rigorous code reviews.
- Focus on improving gas efficiency in existing implementations.
- Help with documentation or writing comprehensive test suites using Foundry or Hardhat.
