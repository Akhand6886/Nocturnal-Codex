### Why contribute to Pekko?
Pekko is the open-source future of the Actor Model on the JVM. Contributing here is a way to work on some of the most complex distributed systems code in the world. You'll learn about location transparency, supervision trees, and cluster sharding.

### Contribution Tips
- **Cluster**: Help improve the reliability of the cluster membership and sharding logic.
- **Streams**: Contribute to the Pekko Streams library for high-throughput data processing.
- **Connectors**: Pekko Connectors (formerly Alpakka) need help integrating with many different external services.
- **Documentation**: As a relatively new fork, updating and improving the documentation is a top priority.
