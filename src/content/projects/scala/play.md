### Why contribute to Play?
Play is the standard MVC framework for the JVM. It's used by companies like LinkedIn and Verizon. Contributing here teaches you about non-blocking I/O, Akka Streams, and how to build a fully developer-friendly web framework with hot-reloading.

### Contribution Tips
- **Java/Scala Interop**: Play supports both languages. Help ensure that new features work seamlessly for both.
- **Akka Streams**: Improve the integration with Akka's reactive stream processing.
- **Security**: Help keep the built-in security features (CSRF, XSS protection) up to date.
- **Templates**: Improve the Twirl templating engine.
