### Why contribute to these samples?
These are the official "Gold Standard" samples from Google. They showcase how to use Jetpack Compose, ViewModel, Room, and Coroutines together. Contributing here helps keep the industry-standard examples up to date with the latest best practices.

### Contribution Tips
- **Jetpack Compose**: Help migrate older View-based samples to modern Jetpack Compose.
- **Testing**: Improve the unit and UI test coverage for the samples.
- **Dependency Injection**: Help improve the Hilt or Koin implementations.
- **Modernization**: Ensure the code follows the latest Kotlin style guidelines.
