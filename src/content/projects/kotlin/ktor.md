### Why contribute to Ktor?
Ktor is JetBrains' native web framework for Kotlin. It is built entirely with Coroutines and DSLs. Contributing here is the best way to master Kotlin's advanced features and learn how to build lightweight, multi-platform HTTP clients and servers.

### Contribution Tips
- **Plugins**: Ktor is highly modular. You can contribute new plugins for authentication, logging, or content negotiation.
- **Samples**: Improving the sample projects is a great way to help other developers.
- **Multiplatform**: Help improve the Ktor Client's support for different platforms (iOS, JS, Desktop).
- **Coroutines**: Master the use of `suspend` functions and `Flow` in a real-world project.
