### Why contribute to Arrow?
Arrow is the premier functional programming library for Kotlin. It brings concepts from Haskell and Scala to the Kotlin world. Contributing here is a deep dive into Category Theory, Typed Error Handling (Either, Option), and effect systems.

### Contribution Tips
- **Typed Errors**: Help improve the `Either` and `Raise` DSLs.
- **Documentation**: Arrow is highly conceptual; contributing clear, visual documentation is incredibly valuable.
- **Optics**: Explore how to improve the `Arrow Optics` library for immutable data manipulation.
- **Analysis**: If you know how to write compiler plugins, help improve the Arrow Analysis tool.
