### Why contribute to EF Core?
EF Core is the standard way to talk to databases in the .NET ecosystem. Contributing here helps you understand how LINQ queries are translated into SQL, how change tracking works, and how to write database providers for PostgreSQL, SQL Server, and SQLite.

### Contribution Tips
- **LINQ**: Improve the translation of specific LINQ patterns into optimized SQL.
- **Providers**: Help fix bugs or add features to specific database providers.
- **Migrations**: Improve the tooling for database schema changes.
- **Logging**: Help improve the observability of database queries.
