### Why contribute to Alamofire?
Alamofire is the most used third-party library in the Apple ecosystem. It's a great place to learn about the internal workings of URLSession, request/response serialization, and modern Swift Concurrency.

### Contribution Tips
- **Swift Concurrency**: Help modernize the library with `async/await` patterns.
- **Documentation**: The library has massive documentation; keeping it up to date with new iOS versions is vital.
- **Testing**: Improving unit tests for network edge cases is always needed.
