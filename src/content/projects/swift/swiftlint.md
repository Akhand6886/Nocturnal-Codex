### Why contribute to SwiftLint?
SwiftLint is essential for any professional Swift project. It uses SourceKit and SwiftSyntax to analyze your code. Contributing here is a great way to learn about Abstract Syntax Trees (AST) and how compilers see your code.

### Contribution Tips
- **New Rules**: If you have an idea for a new code style rule, you can implement it using a simple regex or a complex AST walker.
- **Autocorrect**: Help add "autocorrect" functionality to existing rules.
- **Performance**: Optimize existing rules to make the linter faster on massive projects.
