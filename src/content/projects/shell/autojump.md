### Why contribute to Autojump?
Autojump is a classic shell utility. It's written in Python but integrates deeply with Bash, Zsh, and Fish. Contributing here is a great way to learn about shell hooks and how to write a tool that works across multiple different shells.

### Contribution Tips
- **Shell Integrations**: Help improve the "Hook" logic for specific shells like Fish or PowerShell.
- **Installer**: Improve the installation script for different Linux distributions.
- **Bug Fixes**: Fix edge cases where directory weights are calculated incorrectly.
- **Documentation**: Write a better guide on how to customize the search algorithm.
