### Why contribute to Oh My Zsh?
Oh My Zsh is one of the most starred projects on GitHub. It's a massive community effort. Contributing a new plugin or theme is a fantastic way for a beginner to get their first "Big" open source contribution and learn how Zsh scripting works.

### Contribution Tips
- **New Plugin**: Add an alias or function set for a tool you use daily.
- **Theme**: Design a beautiful, functional Zsh prompt and share it.
- **Bug Fixes**: Help fix broken aliases or installer scripts.
- **Documentation**: Improving the READMEs for the 300+ plugins is a never-ending task.
