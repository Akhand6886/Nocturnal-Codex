### Why contribute to Rakudo?
Rakudo is a compiler for one of the most advanced programming languages in the world. Raku features multiple dispatch, gradual typing, and a powerful grammar system. Contributing here is a deep dive into compiler design, virtual machines, and language architecture.

### Contribution Tips
- **NQP**: Much of Rakudo is written in "Not Quite Perl," a subset of Raku used for bootstrapping.
- **MoarVM**: If you know C, help improve the underlying virtual machine.
- **Roast**: Contribute to the Raku official test suite (the "Roast").
- **Grammar**: Help improve the built-in Raku grammars for parsing complex data.
