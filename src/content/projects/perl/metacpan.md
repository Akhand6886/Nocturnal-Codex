### Why contribute to MetaCPAN?
MetaCPAN is the heart of the Perl ecosystem. It's a large, community-driven project that uses modern Perl, Elasticsearch, and a RESTful API. Contributing here gives you direct visibility in the Perl community.

### Contribution Tips
- **Frontend**: The MetaCPAN web interface uses Catalyst and Template Toolkit.
- **API**: Help improve the backend API that indexes millions of Perl modules.
- **Elasticsearch**: Learn how to write complex search queries for a massive dataset.
- **Grepan**: Help improve the "grep" feature that allows users to search the source code of any CPAN module.
