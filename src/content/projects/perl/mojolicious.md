### Why contribute to Mojolicious?
Mojolicious is a modern masterpiece of Perl engineering. It features a non-blocking I/O loop, a powerful templating engine, and built-in support for WebSockets and JSON. Contributing here is the best way to learn "Modern Perl" and advanced HTTP concepts.

### Contribution Tips
- **Non-blocking**: Master the use of `Mojo::IOLoop` for high-concurrency tasks.
- **CSS Selectors**: Mojo has a built-in CSS selector engine; help improve its performance or standards compliance.
- **Documentation**: Mojolicious has incredible documentation; helping to polish the guides is a great service.
- **Plugins**: Write a new `Mojolicious::Plugin` for a common task like OAuth or Rate Limiting.
