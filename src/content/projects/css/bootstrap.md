### Why contribute to Bootstrap?
Bootstrap is a giant. It has been used by millions of websites for over a decade. Contributing here is a great way to learn about Sass (the CSS preprocessor), accessible component design, and how to maintain a massive library.

### Contribution Tips
- **Accessibility (A11y)**: Help ensure all components are fully WCAG compliant.
- **Sass**: Improve the built-in Sass variables and mixins.
- **Docs**: Help keep the documentation up to date with new browser features.
- **V6 Planning**: Look at the roadmap for the next major version of Bootstrap.
