### Why contribute to Bulma?
Bulma is unique because it contains **Zero JavaScript**. It is purely CSS (based on Flexbox). This makes it an excellent project for developers who want to focus on their CSS skills without getting distracted by a JS framework.

### Contribution Tips
- **Flexbox Mastery**: Help improve the responsiveness of complex layouts using Flexbox.
- **CSS Variables**: Help migrate the library to use modern CSS Custom Properties.
- **Modifiers**: Add new modifier classes for colors, spacing, or typography.
- **Extensions**: Bulma has many community-contributed extensions; check those out too.
