### Why contribute to HTMX?
HTMX is leading the "Return to simplicity" movement in web development. It allows you to build complex UIs without a massive JS framework. Contributing here teaches you about the fundamental power of HTML and how to extend it using custom attributes.

### Contribution Tips
- **Attributes**: Help improve the core `hx-` attributes.
- **Extensions**: HTMX has a small, clean extension system. Help improve the built-in ones or write a new one.
- **Tests**: HTMX has a very clever test suite that simulates server responses. Help improve the coverage.
- **Documentation**: Write a tutorial on how to replace a specific React/Vue feature with a simple HTMX one-liner.
