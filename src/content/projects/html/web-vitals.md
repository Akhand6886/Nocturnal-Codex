### Why contribute to Web Vitals?
Web Vitals are how Google measures the quality of your website. Contributing to this library helps everyone build faster, more responsive sites. You'll learn about low-level browser APIs like `PerformanceObserver` and how to measure things like LCP (Largest Contentful Paint).

### Contribution Tips
- **Metric Accuracy**: Help refine how complex metrics (like CLS - Cumulative Layout Shift) are calculated in edge cases.
- **Polyfills**: Help improve support for browsers that don't yet support the latest performance APIs.
- **Reporting**: Improve how metrics are reported to analytics backends.
- **Tests**: Write complex integration tests that simulate different browser loading behaviors.
