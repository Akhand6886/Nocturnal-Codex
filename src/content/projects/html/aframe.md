### Why contribute to A-Frame?
A-Frame brings VR to the web using just HTML. It uses an Entity-Component architecture. Contributing here is the best way to learn about 3D graphics (Three.js), WebXR, and how to build high-performance visual experiences in the browser.

### Contribution Tips
- **Components**: Write a new A-Frame component for a specific interaction (like teleporting or object physics).
- **Documentation**: A-Frame is very visual; help improve the guides with better diagrams and live examples.
- **WebXR**: Help improve support for the latest VR and AR headsets.
- **Performance**: Optimize the rendering loop to ensure a smooth 90 FPS experience in VR.
