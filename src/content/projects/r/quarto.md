### Why contribute to Quarto?
Quarto is the future of scientific communication. It's a massive project that combines Pandoc, Lua, and various language kernels. Contributing here is a way to work on a truly polyglot tool used by the world's leading researchers.

### Contribution Tips
- **Lua Filters**: If you know Lua, writing custom Pandoc filters is a great way to extend Quarto.
- **Extensions**: Contribute to the Quarto extensions ecosystem (themes, shortcodes, journals).
- **CLI**: The main CLI is written in TypeScript; help improve the build and preview logic.
- **Languages**: Improve how Quarto interacts with specific language kernels (like Jupyter or Knitr).
