### Why contribute to Hugo?
Hugo is famously fast because it's written in Go. Contributing to it gives you experience with Go's templating system, concurrency (for fast builds), and how to handle massive amounts of file-based data.

### Contribution Tips
- **Template Functions**: Add new built-in functions to make life easier for theme developers.
- **Performance**: Help find even more ways to shave milliseconds off the build time.
- **Documentation**: Improving the documentation for complex features like "Shortcodes" or "Image Processing" is very helpful.
