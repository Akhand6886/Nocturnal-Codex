### Why contribute to Supabase?
Supabase is proof that "Everything is just Postgres." Contributing here is a great way to see how modern web features (like Auth and Realtime) can be implemented using SQL functions, triggers, and Row Level Security (RLS).

### Contribution Tips
- **SQL Functions**: Help improve the built-in SQL functions and triggers.
- **Client Libraries**: If you know JS, Python, or Swift, contribute to the SDKs that talk to the Postgres backend.
- **Dashboard**: Help improve the SQL editor and table view in the Supabase Studio.
- **Documentation**: Write a tutorial on how to implement a specific business logic using only SQL.
