### Why contribute to TimescaleDB?
TimescaleDB is one of the most successful PostgreSQL extensions. It handles billions of data points for IoT and monitoring. Contributing here teaches you about "Hypertables," automatic partitioning, and high-performance data compression.

### Contribution Tips
- **Continuous Aggregates**: Help improve how the database pre-calculates results for common time-based queries.
- **Compression**: Explore how the database uses columnar storage to save space.
- **Query Planner**: Help optimize the query planner for time-bucketed operations.
- **Observability**: Add more metrics and telemetry to help users monitor their hypertables.
