### Why contribute to Actix-web?
Actix-web is consistently one of the fastest web frameworks in the world. It's built on top of the Actix actor system. Contributing here helps you learn about high-performance networking, futures, and asynchronous programming in Rust.

### Contribution Tips
- **Middleware**: Write reusable middleware for things like session management or rate limiting.
- **Examples**: The `actix/examples` repo is a great place to contribute small, focused use cases.
- **Tests**: Help maintain the performance and stability of the framework with better unit and integration tests.
