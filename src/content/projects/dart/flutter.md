### Why contribute to Flutter?
Flutter is one of the fastest-growing open-source projects. Contributing here allows you to shape the future of multi-platform development. You can work on anything from the Skia rendering engine to the high-level Widget library.

### Contribution Tips
- **Samples**: Adding high-quality examples to the `flutter/samples` repo is a great way to start.
- **Widgets**: Help improve the "Material" or "Cupertino" widget sets.
- **Packages**: Many core Flutter features live in separate packages (like `path_provider` or `camera`). Check those repos too.
- **Documentation**: Fixing a confusing doc comment in the source code is very helpful.
