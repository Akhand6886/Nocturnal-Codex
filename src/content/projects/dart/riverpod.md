### Why contribute to Riverpod?
Riverpod is the current community favorite for state management in Flutter. It's written with extreme care for type-safety and developer experience. Contributing here helps you learn advanced Dart features like mixins, generics, and code generation.

### Contribution Tips
- **Documentation**: Riverpod has a dedicated documentation site. Helping improve the guides or examples is highly valued.
- **Riverpod Generator**: If you know how to write build_runner plugins, help improve the code generation part.
- **Bug Reports**: Provide high-quality reproduction cases for reported bugs.
