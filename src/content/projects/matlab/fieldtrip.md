### Why contribute to FieldTrip?
FieldTrip is a massive, well-organized MATLAB toolbox for brain data analysis. It is built by researchers at the Donders Institute. Contributing here is a great way to learn advanced signal processing and how to manage a huge collection of MATLAB functions.

### Contribution Tips
- **Algorithms**: Implement a new signal processing algorithm or improve an existing one.
- **Tutorials**: FieldTrip relies heavily on community-contributed tutorials. Writing a new one for a specific data type is high-impact.
- **Bug Fixes**: With such a large codebase, there are always edge cases to fix in data importing and preprocessing.
- **Visualisation**: Help improve the built-in 2D and 3D brain plotting functions.
