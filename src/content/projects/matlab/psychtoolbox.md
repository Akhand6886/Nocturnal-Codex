### Why contribute to Psychtoolbox?
Psychtoolbox is the standard for visual perception research. It interacts directly with graphics hardware and low-level system APIs. Contributing here teaches you about high-precision timing, OpenGL, and cross-platform hardware interfacing in MATLAB.

### Contribution Tips
- **GStreamer**: Help improve the video playback engine.
- **Hardware Support**: Add support for new eye-trackers or input devices.
- **Demos**: Improve the built-in demo scripts that show new researchers how to use the toolbox.
- **Linux/Mac/Win**: Help ensure the low-level MEX files work perfectly across all operating systems.
