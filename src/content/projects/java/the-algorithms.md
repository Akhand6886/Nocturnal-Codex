### Why contribute to The Algorithms?
This is a purely educational project. It's the perfect place to contribute your first piece of code by implementing a missing algorithm or improving an existing one.

### Contribution Tips
- **New Algorithms**: Check the issues for requests for new algorithm implementations.
- **Refactoring**: Help make the existing code more readable or efficient.
- **Documentation**: Adding explanations for *how* an algorithm works in the code comments.
