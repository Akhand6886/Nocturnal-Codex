### Why contribute to Guava?
Guava is the most used Java utility library. It's where many of Java's core features (like Optional or Streams) were pioneered. It's a masterclass in Java performance and collection design.

### Contribution Tips
- **Performance**: Many issues involve optimizing existing collection implementations.
- **Javadoc**: Improving the Javadoc clarity is a great way to start.
- **Tests**: Helping to modernize the test suite.
