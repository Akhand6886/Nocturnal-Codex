### Why contribute to Guzzle?
Guzzle is the industry standard for making HTTP requests in PHP. It's used by almost every major PHP application. Contributing here helps you understand the nuances of the HTTP protocol, promises, and asynchronous streams.

### Contribution Tips
- **HTTP/2**: Help improve support for the latest web protocols.
- **Handlers**: Explore how Guzzle handles different cURL configurations.
- **Middleware**: Guzzle uses a middleware system; learning how to write these is a great entry point.
