TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale. It's a "superset" of JavaScript, meaning any valid JavaScript code is also valid TypeScript code. By adding types, you can catch errors during development that you would otherwise only find at runtime.
