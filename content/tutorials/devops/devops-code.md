## 2. The Code Stage

The **Code** stage is where the development happens. Developers write code to implement the features defined in the planning stage.

### Key Activities:
- **Version Control:** Using a system like **Git** to track changes to the codebase. This is crucial for collaboration, as it allows multiple developers to work on the same project without overwriting each other's work.
- **Branching Strategies:** Using strategies like GitFlow to manage features, releases, and hotfixes in a structured way.
- **Code Quality:** Writing clean, maintainable, and well-documented code.

This stage is all about collaborative software development and maintaining a single source of truth for the codebase.
