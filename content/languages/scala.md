Scala is a strong statically typed general-purpose programming language that supports both object-oriented and functional programming. Designed to be concise, many of Scala's design decisions are aimed to address criticisms of Java. Scala source code is intended to be compiled to Java bytecode, so that the resulting executable code runs on a Java virtual machine (JVM).
